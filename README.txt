The dataset includes the following files:
=========================================

- 'README.txt'

- 'codebook.txt': Shows information about the variables used.


- 'run_analysis.R': Contains the analysis that compiles all of the data, selects just the means and standard deviations, renames columns to a readable format, and computes the means for each subject and for each activity.
